* emt.el

** Introduction

A fork of [[https://github.com/roife/emt][EMT]] with cross-platform Chinese word segmentation powered by [[https://github.com/messense/jieba-rs][jieba-rs]].

Instead of the original macOS-only Swift module (or the ICU-based ewt-rs), this fork uses a built-in Rust dynamic module with jieba-rs, providing accurate Chinese word segmentation on all platforms:

- macOS (aarch64, x86_64)
- Linux (x86_64)
- Windows (x86_64)

** Installation

*** Requirements

- Emacs 26.1 or later, built with dynamic module support (use =--with-modules= during compilation)
- macOS, Linux, or Windows

*** Dynamic module

**** Pre-built (recommended)

If you enable =emt-mode= and the module cannot be found, it will prompt you to download a pre-built module from GitHub or build from source. You can also manually call =emt-download-module= to download it. The module is placed at =emt-lib-path= (by default =~/.emacs.d/modules/libEMT.<suffix>=).

**** Build from source (optional)

Requires the [[https://rustup.rs/][Rust toolchain]]. Build manually from the =module/= directory in the repo:

#+begin_src shell
  cd module
  cargo build --release
  cp target/release/libemt_module.dylib ~/.emacs.d/modules/libEMT.dylib  # macOS
  # cp target/release/libemt_module.so ~/.emacs.d/modules/libEMT.so      # Linux
#+end_src

*** Install package

Install with =straight= and =use-package=:

#+begin_src emacs-lisp
  (use-package emt
    :straight (:host github :repo "LuciusChen/emt"
                     :files ("*.el"))
    :hook (after-init . emt-mode))
#+end_src

** Customization

*** =emt-use-cache=

Caches for results of tokenization if non-nil. Default is =t=.

*** =emt-cache-lru-size=

The size of LRU cache. Default is =50=.

*** =emt-lib-path=

The path to the dynamic library for emt. Default is =~/.emacs.d/modules/libEMT<suffix>=.

** Usage

*** keymap: =emt-mode-map=

It remaps =forward-word=, =backward-word=, =kill-word= and =backward-kill-word= to use emt's version.

*** Minor mode

It calls =emt-ensure=, which loads the dynamic module and sets =emt-mode-map=.

*** Functions

**** =emt-word-at-point-or-forward=

Return the word at point. If current point is at bound of a word, return the one forward.

**** =emt-word-at-point-or-backward=

Return the word at point. If current point is at bound of a word, return the one backward.

**** =emt-download-module=

Download the pre-built dynamic module from GitHub releases.

**** =emt-build-module=

Build the dynamic module from source using cargo.

**** =emt-ensure=

Ensure the dynamic module is loaded, downloading or building it if necessary.

**** =emt-split=

Split string into a list of words.

Return a list of word bounds (a cons of the beginning position and the ending position of a word)

**** =emt-forward-word=

CJK compatible version of =forward-word=.

**** =emt-backward-word=

CJK compatible version of =backward-word=.

**** =emt-kill-word=

CJK compatible version of =kill-word=.

**** =emt-backward-kill-word=

CJK compatible version of =backward-kill-word=.

**** =emt-mark-word=

CJK compatible version of =mark-word=.
