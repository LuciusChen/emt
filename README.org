* emt.el

** Introduction

A fork of [[https://github.com/roife/emt][EMT]] with cross-platform support via [[https://github.com/Master-Hash/ewt-rs][ewt-rs]].

Instead of the original macOS-only Swift module, this fork downloads the ewt-rs dynamic module which supports:

- macOS (ICU backend)
- Linux (ICU backend)
- Windows (WinRT backend, recommended; or ICU)

** Installation

*** Requirements

- Emacs 26.1 or later, built with dynamic module support (use =--with-modules= during compilation)
- macOS, Linux, or Windows

*** Dynamic module

**** Pre-built (recommendation)

If you enable =emt-mode= and the module cannot be found, it will prompt whether to automatically download the ewt-rs pre-built module from GitHub. Or you can manually call =emt-download-module= to download it. The module is placed at =emt-lib-path= (by default =~/.emacs.d/modules/libEMT.<suffix>=).

*Current version of the ewt-rs module is v0.4.0.*

*** Install package

Install with =straight= and =use-package=:

#+begin_src emacs-lisp
  (use-package emt
    :straight (:host github :repo "LuciusChen/emt"
                     :files ("*.el"))
    :hook (after-init . emt-mode))
#+end_src

** Customization

*** =emt-use-cache=

Caches for results of tokenization if non-nil. Default is =t=.

*** =emt-cache-lru-size=

The size of LRU cache. Default is =50=.

*** =emt-lib-path=

The path to the dynamic library for emt. Default is =~/.emacs.d/modules/libEMT<suffix>=.

*** =emt-windows-use-winrt=

If non-nil, download the smaller WinRT backend on Windows instead of ICU. WinRT is recommended for Simplified Chinese; ICU for Traditional Chinese. Default is =nil=.

** Usage

*** keymap: =emt-mode-map=

It remaps =forward-word=, =backward-word=, =kill-word= and =backward-kill-word= to use emt's version.

*** Minor mode

It calls =emt-ensure=, which load dynamic modeuls and set =emt-mode-map=.

*** Functions

**** =emt-word-at-point-or-forward=

Return the word at point. If current point is at bound of a word, return the one forward.

**** =emt-word-at-point-or-backward=

Return the word at point. If current point is at bound of a word, return the one backward.

**** =emt-download-module=

Download the ewt-rs dynamic module from GitHub releases. Supports macOS, Linux, and Windows. If PATH is non-nil, download the module to PATH.

**** =emt-ensure=

Ensure the dynamic module is loaded, downloading it if necessary.

**** =emt-split=

Split string into a list of words.

Return a list of word bounds (a cons of the beginning position and the ending position of a word)

**** =emt-forward-word=

CJK compatible version of =forward-word=.

**** =emt-backward-word=

CJK compatible version of =backward-word=.

**** =emt-kill-word=

CJK compatible version of =kill-word=.

**** =emt-backward-kill-word=

CJK compatible version of =backward-kill-word=.

**** =emt-mark-word=

CJK compatible version of =mark-word=.
